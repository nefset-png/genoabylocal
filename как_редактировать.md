# Genoa Local Experiences — Инструкция

## Как поменять фотографии

1. Создай папку `images/` рядом с файлом `index.html`
2. Положи туда свои фото с этими именами:

### Главная страница
- `hero.jpg` — большое фото в герое (рекомендуется 1400×900px)
- `about.jpg` — твоё фото в секции "Hi, I'm your local host"
- `tour-genoa.jpg` — фото карточки тура Genoa
- `tour-portofino.jpg` — фото карточки тура Portofino
- `tour-ct.jpg` — фото карточки тура Cinque Terre

### Тур Genoa (слайдер + галерея)
- `genoa-1.jpg` до `genoa-5.jpg`

### Тур Portofino (слайдер + галерея)
- `portofino-1.jpg` до `portofino-5.jpg`

### Тур Cinque Terre (слайдер + галерея)
- `ct-1.jpg` до `ct-5.jpg`

---

## Как поменять текст

Открой `index.html` в текстовом редакторе.
Используй поиск (Ctrl+F) чтобы найти нужный текст и замени его.

---

## Как подключить Calendly

Найди в коде:
```
href="https://calendly.com/YOUR-LINK"
```
Замени `YOUR-LINK` на твою ссылку Calendly.

---

## Как опубликовать бесплатно

### GitHub Pages
1. github.com → создай аккаунт
2. New repository → назови `genoa-tours`
3. Upload files → загрузи `index.html` и папку `images/`
4. Settings → Pages → Source: main
5. Сайт: `твой-логин.github.io/genoa-tours`

### Bolt.new
1. bolt.new
2. Вставь содержимое `index.html`
3. Редактируй через чат
